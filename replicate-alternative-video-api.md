## Technical Integration: Getting Started with kinovi.ai API

For developers choosing kinovi.ai, here's the integration workflow:

### Step 1: Get API Key
Sign up for a paid plan (Starter minimum) to receive API credentials from your kinovi.ai dashboard.

### Step 2: Make Your First Request
```bash
# Submit video generation job
curl -X POST "https://kinovi.ai/api/v1/video/generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene mountain landscape at sunset",
    "model": "seedance-2.0-fast",
    "duration": 5,
    "resolution": "720p"
  }'
```

### Step 3: Poll for Completion
```bash
# Check job status using returned taskId
curl "https://kinovi.ai/api/v1/jobs/recordInfo?taskId=your_task_id" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

The platform supports concurrent requests (2-50 parallel jobs depending on plan[[105]](https://seedance2-pro.com/pricing)) and returns video URLs upon completion. For production deployments, implement webhook callbacks instead of polling to reduce API calls.

### Code Example: Python Integration
```python
import requests
import time

API_KEY = "your_kinovi_api_key"
BASE_URL = "https://kinovi.ai/api/v1"

def generate_video(prompt, duration=5, resolution="720p"):
    # Submit job
    response = requests.post(
        f"{BASE_URL}/video/generate",
        headers={"Authorization": f"Bearer {API_KEY}"},
        json={
            "prompt": prompt,
            "duration": duration,
            "resolution": resolution,
            "model": "seedance-2.0-pro"
        }
    )
    task_id = response.json()["taskId"]
    
    # Poll until complete
    while True:
        status = requests.get(
            f"{BASE_URL}/jobs/recordInfo",
            params={"taskId": task_id},
            headers={"Authorization": f"Bearer {API_KEY}"}
        )
        if status.json()["status"] == "completed":
            return status.json()["videoUrl"]
        time.sleep(5)
```
