# about-materials

把你的个人资料放这里，用来重写首页 About section 的文案。

## 需要的资料

1. **开发经历** — 你在哪些领域做过开发？（语言、框架、项目类型）
2. **Agents 领域** — 你在 AI Agents 方面做了什么？（工具、自动化链路、产品）
3. **转折故事** — 为什么从开发转向 SEO？为什么用工程方法做 SEO？
4. **关键里程碑** — 第一个客户、第一个大项目、最有成就感的事
5. **个人风格** — 你想让访客感受到什么？（可以直接丢几段你觉得"像我"的文字）
6. **照片/素材** — 任何你想放到 About section 的图片

格式不限，语音转文字、碎片笔记、甚至语音文件名都可以。放进来告诉我一声就行。
